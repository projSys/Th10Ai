# TH10Bot

东方机器人

## 闪避规则

1、自机与子弹前进路径的垂足距离最近，只要与垂足不碰撞，那么与路径上的其他点都不会碰撞。

2、由此可以计算得到某一坐标点的帧轴（Z轴），哪一帧是安全的，哪一帧是危险的。
